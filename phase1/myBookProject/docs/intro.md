# Introduction

![ROS 2 Architecture](/img/robot.jpg)

Welcome to the ROS 2 Nervous System for Physical AI & Humanoid Robotics course. This module will teach you how to establish ROS 2 as the foundational communication and control layer that connects AI decision-making systems to humanoid robot bodies in simulation and real-world environments.

## What You'll Learn

- ROS 2 fundamentals: nodes, topics, and services
- How to connect Python AI agents with rclpy
- How to define robot bodies with URDF for humanoid robotics
- Best practices for creating a robotic nervous system
